# AI Studio Web Starter 🚀

A professional, performance-optimized starter template for building modern AI-powered web applications. Built for the Google AI Studio ecosystem, this repository provides a solid foundation with React, Tailwind CSS, and the Gemini API.

## ✨ Features

- **⚛️ React 19** - The latest features for high-performance UI.
- **🎨 Tailwind CSS 4** - Modern utility-first styling with zero-runtime CSS.
- **🤖 Gemini AI Integration** - Pre-configured `@google/genai` SDK for building smart features.
- **✨ Framer Motion** - Smooth, professional animations and transitions.
- **🛠️ Vite** - Lightning-fast build tool and development server.
- **📦 Lucide React** - Clean and consistent iconography.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- A Gemini API Key (get one at [aistudio.google.com](https://aistudio.google.com/app/apikey))

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   cd ai-studio-web-starter
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Configure your API Key:
   Create a `.env` file in the root directory and add your key:
   ```env
   GEMINI_API_KEY=your_actual_api_key_here
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:3000`.

## 🏗️ Project Structure

- `src/` - Source code and components.
- `src/App.tsx` - Main application entry point.
- `src/index.css` - Global styles and Tailwind configuration.
- `metadata.json` - AI Studio specific metadata.
